# 隐私政策（Privacy Policy）

更新日期：2026-01-14

本隐私政策适用于 Chrome 扩展程序「GMGN 同名首发过滤器（GMGN Same-Name First Launch Filter）」（以下简称“本扩展”）。本扩展旨在仅在 gmgn.ai 页面上标记同名代币的“首发/非首发”并提供跳转首发代币页面的快捷入口。

## 1. 我们收集哪些数据

本扩展不收集、不过滤并不向任何服务器上传任何可识别个人身份的信息（PII），包括但不限于：姓名、邮箱、手机号、地址、身份证号、IP 地址、精确位置、浏览历史、键盘输入、钱包私钥/助记词等。

本扩展仅会在用户访问 https://gmgn.ai/* 时读取页面中已公开显示的内容（例如代币列表中的 Symbol/Name/age/链接等），用于在页面内进行“首发/非首发”标记与按钮展示。此类页面内容不会被本扩展持久化存储，也不会被传输到任何外部服务器。

## 2. 我们如何使用数据

本扩展读取 gmgn.ai 页面公开信息的目的仅限于：
- 在列表中标记“首发/非首发”
- 在非首发代币上显示“跳转”按钮并打开对应首发代币页面
- 根据用户设置的时间窗口/匹配模式决定标记逻辑

本扩展不会将任何信息用于广告投放、用户画像、行为分析、追踪或营销。

## 3. 本地存储（Storage）说明

本扩展使用 Chrome Storage（chrome.storage.sync 或 chrome.storage.local）仅用于保存用户的功能设置，例如：
- 语言（中文/English/自动）
- 时间窗口（分钟）
- 匹配模式（默认仅 Symbol 相同）
- 是否仅在时间窗口内比较
- 其他与扩展功能相关的设置项

这些设置数据仅用于在用户刷新页面或跨设备同步时保持配置一致，不会包含 gmgn.ai 页面内容，不会包含任何个人身份信息，也不会被上传至开发者服务器或第三方服务器。

## 4. 数据共享与第三方

- 本扩展不会向第三方出售、共享或转移任何用户数据。
- 本扩展不集成第三方分析/统计 SDK，不集成广告 SDK。

## 5. 网络请求与远程代码

- 本扩展不会向开发者服务器发起任何网络请求，也不会上传任何数据。
- 本扩展不使用远程代码：不加载或执行任何未包含在扩展安装包内的 JavaScript/Wasm，不使用外部脚本引用，不通过 eval() 动态执行字符串代码。
- “跳转”按钮仅会打开 gmgn.ai 的对应 token 页面链接（在新标签页中打开）。

## 6. 数据安全

本扩展不存储敏感数据、不上传数据，且所有逻辑均包含在扩展包内。用户设置仅保存在浏览器的扩展存储中。

## 7. 儿童隐私

本扩展不面向儿童，不会主动收集任何儿童相关信息。

## 8. 隐私政策变更

如本扩展的功能或数据处理方式发生变化（例如新增数据收集/分析/远程服务等），我们将更新本隐私政策并在版本更新时同步说明。
